# Maincrafts Task 4 - Task Manager Dashboard

## 📌 Project Overview
This is my Maincrafts Internship Task 4 project. It is a Task Manager Dashboard built using HTML, CSS, and JavaScript.

## 🚀 Features
- ✅ Add Task
- ✅ Edit Task
- ✅ Delete Task
- ✅ Search Tasks
- ✅ Mark Task as Completed
- ✅ Task Counter (Total, Completed, Pending)
- ✅ Local Storage Support
- ✅ Responsive & Premium UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- VS Code
